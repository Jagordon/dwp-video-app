Hypotheses

Because:

  • research shows some users do not know which browser they're using and whether it's supported
We believe that:
  • giving information about supported browsers with links to download them and allowing the user to check which one they're currently using
Will result in:
  • the user understanding whether they have a supported browser and be able to download one if they don't
We'll know this is true:
  • when we have less users unable to join the meeting because they're on the wrong browser

Because:

  • we want feedback about the quality of the service to improve it
  • we know the best time to capture feedback is straight after using the service
We believe that:
  • providing a short survey on the end page
Will result in:
  • feedback about the quality of the service from the majority of users
We'll know this is true:
  • when a high percentage of users complete the survey

Because:

  • we may have Welsh speakers using our service
We believe that:
  • giving the option to switch to Welsh on the start page and keeping it throughout the journey
Will result in:
  • the citizen using the service in the langauge they prefer
We'll know this is true:
  • if Welsh speakers are able to use the service in their preferred language
