Version 2

Assumptions:

  • Welsh speaking users will need the service displayed in Welsh.
  • Users will want to give feedback on the quality of their video appointment as soon as it is finished.
  • Users need to understand what browser they are on so they know whether they will be able to access their video appointment.
Version details: 2.0

2.0

  • Welsh toggle added to start page. This takes the user to the welsh version of the service.
  • Welsh toggle added to header of screens not accessed via service start page.
  • Short feedback survey added to the end page for the claimant. If they hit 'Send feedback' they are taken to the thank you screen.
  • Thank you screen has a link to a citizen space survey where they can give their details to be involved in further research.

2.1

  • Content of help and guidance page updated to include more specific information about which browsers and versions are supported.
  • Link out to whatsmybrowser.org to tell the user which browser and version they are currently on to see if it is supported.
Hypotheses

Because:

  • we need attendee feedback about the service
  • we know the best time to capture that feedback is as soon as possible
  • we know that this feedback is not mandatory to the user using the service
We believe that:
  • giving a short survey on the current end page of the service
Will result in:
  • a high number of attendees giving their accurate feedback
We'll know this is true:
  • when we have a high percentage of users reaching the 'thank you' page

Because:

  • the trial has shown some users do not know which browser they are using and whether it is supported
We believe that:
  • by giving specific information about which browsers are supported within the help and guidance and allowing the user to check which they are currently on
Will result in:
  • the user understanding whether they will be able to have a good-quality meeting on a supported browser before joining
We'll know this is true:
  • when we have fewer attendees unable to join the meeting because they are on the wrong browser.

Because:

  • we know that we will have Welsh speakers using our service
We believe that:
  • by giving the option to switch to Welsh language at the start of the journey
Will result in:
  • the citizen completing their appointment in the langauge they prefer
We'll know this is true:
  • Welsh speakers have a successful appointment.
