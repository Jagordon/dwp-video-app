Your camera or microphone is blocked

You will need to unblock access to your camera and microphone to take part in the video appointment. Follow the instructions for your device and browser.

Mobile

Safari on iPhone or iPad

  1. Select the 'AA' icon in the address bar.
  2. Select 'Website settings'.
  3. Select 'Camera' or 'Microphone' and set to 'Allow'.
  4. Select 'Done' and reload the page.

Chrome on Android

  1. Select the 3 dots in the browser toolbar.
  2. Select 'Settings' then 'Site settings'.
  3. Select 'Camera' or 'Microphone'.
  4. Make sure 'jobcentre.dwp.gov.uk' is in the 'Allowed' list.
  5. If it is in 'Blocked', select it and set the camera or microphone to 'Allow'.
  6. Exit the settings and reload the page.

Desktop

Safari

  1. Select 'Safari' on your menu bar.
  2. Select 'Settings for this website'.
  3. Select 'Allow' for 'Camera' or 'Microphone'.
  4. Exit the settings and reload the page.

Chrome

  1. Select the lock icon in the address bar.
  2. Select 'Allow' next to 'Camera' or 'Microphone'.
  3. Exit the settings and reload the page.
