Hypotheses

Because:

  • we know hosts want a professional start to the appointment
  • we know hosts are not always the first to join an appointment
We believe that:
  • letting users preview their video before joining the appointment
Will result in:
  • a smoother start to the appointment
  • work coaches feeling more comfortable joining the appointment
We'll know this is true:
  • when users give us feedback that the extra step of previewing their video is beneficial.

Because:

  • we know when some users join the meeting first, they use the refresh button multiple times before the other person joins
  • we know some users do not like seeing the large view of themselves as it makes them uncomfortable
We believe that:
  • by explaining they have joined the appointment, but the other person hasn't
  • by making the large view of themselves a smaller 'selfie' view
Will result in:
  • users understanding what is happening and that they do not need to do anything
  • users feeling more comfortable
We'll know this is true:
  • when users give us feedback that they understand what is happening
  • when users give us feedback that they feel more comfortable with a smaller 'selfie' view
  • if there is a reduction in the use of the refresh button prior to the second person joining

Version 3

Link access

Code access

Hypotheses

Because:

  • we know from research that work coaches find a link easier and quicker
  • there is a need for work coaches to be time efficient
We believe that:
  • providing a link in to a specific appointment
Will result in:
  • a simpler and shorter journey
We'll know this is true:
  • when work coaches join an appointment faster.
If we still provide a start page in the link journey, the work coach will be able to prepare (for example arranging their screen) and have a more professional start to the appointment.

If we show a smaller view of themselves than the other participants, work coaches will feel at ease and be able to focus on the other participants in the appointment.
If we show the whole frame of other participants, work coaches will be able to see everything they need to see.

Version 2

Hypotheses

Because:

  • there are security concerns around uninvited attendees being able to enter the appointment
  • we have observed attendees trying to access their appointment from multiple devices
We believe that:
  • displaying a list of participants in the call to the host
Will result in:
  • a higher level of security
  • greater control of the appointment
  • fewer issues with video feeds
We'll know this is true:
  • when observations show that work coaches are able to identify when participant(s) are in the call
  • when feedback shows the work coach has investigated any unexpected participants and taken the appropriate action.
