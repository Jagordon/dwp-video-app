Your camera or microphone is blocked

You must unblock access to your camera and microphone to take part in the video appointment. Follow the instructions for your device and browser.

Mobile

Chrome on Android

  1. Select the 3 dots in the browser toolbar.
  2. Select 'Settings'.
  3. Select 'Site settings'.
  4. Select 'Camera' or 'Microphone'.
  5. Make sure this website is in the 'Allowed' list.
  6. If it is in 'Blocked', select it and set the camera or microphone to 'Allow'.
  7. Exit the settings and continue from this page.

Safari on iPhone or iPad

  1. Select the 'AA' icon in the address bar.
  2. Select 'Website settings'.
  3. Select 'Camera' or 'Microphone' and select 'Allow'.
  4. Select 'Done' and continue from this page.

Desktop

Safari

  1. Select 'Safari' on your menu bar.
  2. Select 'Settings for this website'.
  3. Select 'Allow' for 'Camera' or 'Microphone'.
  4. Exit the settings and continue from this page.

Chrome

  1. Select the lock icon in the address bar.
  2. Select 'Allow' for 'Camera' or 'Microphone'.
  3. Exit the settings and continue from this page.
