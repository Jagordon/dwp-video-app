Combined testing link 1 - March 2021
Hypotheses

Small selfie

Because: we know not all users feel comfortable seeing themselves on screen and need a clear view of other participants
We believe that: showing a small view of themselves in the corner of the screen
Will result in: users feeling at ease and able to focus on the other participants in the appointment
We'll know this is true: when users state this in user testing.

Equal selfie

Because: users need to ensure they can be seen clearly
We believe that: showing all feeds, including themselves, in a grid format
Will result in: users being seen clearly throughout the appointment
We'll know this is true: when users choose this option in user testing over the alternatives.

Fill frame

Because: we know people value seeing facial expressions
We believe that: showing a cropped, zoomed in view of other participants
Will result in: attendees being able to read facial expressions and an efficient use of screen space
We'll know this is true: when users choose this option in user testing over the alternatives.

Fit-to-frame

Because: we know users sometimes need to see the whole frame
We believe that: keeping the whole frame of other participants in view
Will result in: users being able to see everything they need to see
We'll know this is true: when users choose this option in research over the alternatives.

Blur background

Because: there is a need for users to hide their background
We believe that: blurring the background
Will result in: sufficient privacy for the user
We'll know this is true: when users state this in user testing.

Cover background

Because: there is a need for users to hide their background
We believe that: fully replacing the background with a pre-selected image
Will result in: greater privacy for the user
We'll know this is true: when users choose this option in research.

Radio buttons

Because: there is a need for users to hide their background
We believe that: providing radio buttons once in the appointment
Will result in: understanding whether blur is on or off, and how to change it
We'll know this is true: when participants show their understanding in usability testing.

Settings drop down menu

Because: we know there is a need for a citizen to hide their background and we assume the action will only be once per appointment, if at all
We believe that: providing a settings menu
Will result in: a simple interface, with less distractions from the appointment
We'll know this is true: when participants show their understanding in usability testing.
